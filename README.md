# DLB-Pytorch

A minimalist unofficial implementation of "Self-Distillation from the Last Mini-Batch for Consistency Regularization"

reference: https://arxiv.org/pdf/2203.16172.pdf

## Prerequisites

+ PyTorch
+ PyTorch Lightning
+ timm
+ torchmetrics
+ torchvision
+ python3
+ CUDA
